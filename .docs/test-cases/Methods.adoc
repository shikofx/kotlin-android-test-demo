[cols="1,3,2,2,2"]
|===
| ID | Scenario | Input Data (`str1`, `str2`) | Expected Result | Test Type

| *ISE-1* | Happy Path (identical strings) | `"hello"`, `"hello"` | `true` | Positive

| *ISE-3* | Negative (different strings) | `"hello"`, `"world"` | `false` | Negative
